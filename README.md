# MatrixLib

MatrixLib 是 Matrix 系列插件的共享前置插件，用于统一品牌风格、控制台加载动画、文本前缀、YAML 消息读取和资源文件操作。

它的目标是把 MatrixShop、MatrixAuth、MatrixCook 中重复的基础能力抽离出来，减少重复实现，保证后续功能和风格迭代可以集中维护。

## 适用范围

- 统一 Matrix 系列插件的文本与控制台视觉风格
- 提供共享的消息、资源和辅助 API
- 作为其他 Matrix 插件的前置依赖
